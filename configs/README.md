# config layout

These files are the shared contract for the first implementation passes.

- `env/` holds benchmark and path-level config
- `train/` holds method configs for RBC, PPO, and SAC
- `eval/` holds evaluation output settings
- `splits/` holds split names and intent, not downloaded data

The files here are structural and documented on purpose. They are not wired into a real CityLearn loader yet.
